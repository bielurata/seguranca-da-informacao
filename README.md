# Segurança da Informação Labs

Repositório contendo laboratórios e projetos desenvolvidos durante minha graduação em Segurança da Informação e meus estudos no curso Novo Pentest Profissional (NPP) da Desec Security.

## Tecnologias

* Linux
* Bash
* Linguagem C
* TCP/IP
* DNS
* Nmap

## Projetos

### Bash

* pingsweep.sh
* knock.sh
* parsing.sh

### Linguagem C

* resolverdns.c
  
## Como executar

### pingsweep.sh

chmod +x pingsweep.sh -> 
./pingsweep.sh

### parsing.sh

chmod +x parsing.sh -> 
./parsing.sh https://exemplo.com

### resolverdns.c

gcc resolverdns.c -o resolverdns -> 
./resolverdns google.com

## Objetivo

Praticar conceitos de redes, automação, reconhecimento de ambientes e programação de redes.

## Aprendizados

Durante o desenvolvimento destes laboratórios pratiquei:

- Reconhecimento de redes
- Descoberta de hosts
- Enumeração de serviços
- Resolução DNS
- Programação de redes em C
- Automação com Bash
- Utilização do Nmap
- Conceitos de TCP/IP

## Aviso

Todos os projetos foram desenvolvidos para fins educacionais em ambientes controlados e autorizados.
